-------------------------------------------

1 입력
이름 / 비밀번호 / 메모 형식으로 입력해주세요. => split / 각각을 필드에 넣어줘야함

메모: 
-------------------------------------------
2 목록 보기
메모리스트
1 잠자기 (성루비, 2022-11-23)

3 잠자기 (성루비, 2022-11-23)
-------------------------------------------
3 수정
입력 > 5
글이 존재하지 않습니다.
입력 > 1

-------------------------------------------
4 삭제
입력 > 5

입력 > 1

작성일(컴퓨터 시스템의 날짜와 시간을 자동으로 저장) date - string
